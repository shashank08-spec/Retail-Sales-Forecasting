Retail Sales Forecasting

This project looks at UK retail transaction data to understand sales trends, seasonal changes, and predict future sales using time series methods.


Dataset Used:
- Online Retail Dataset from the UK
- Includes detailed transaction information such as timestamps, quantities, and prices

Approach Taken:
- Loaded and cleaned the data
- Created datetime features to better understand time patterns
- Conducted exploratory data analysis to uncover patterns
- Aggregated sales data on a monthly basis
- Used ARIMA, a time series forecasting method, to predict future sales

Key Findings:
- Sales follow a seasonal pattern throughout the year
- There is a difference in sales between weekdays and weekends
- ARIMA model gives a decent estimate for future sales

Tools Used:
- Python
- Pandas
- Matplotlib
- Statsmodels
- Google Colab
